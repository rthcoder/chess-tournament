# Chess Tournament Management System

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/tohirdavrsmart/chess-tournament.git
   cd chess-tournament
   ```

2. Install dependencies:
``` npm install ```

4. Create a .env file with the following content:
     ```
    PORT=
    MONGO_URI=
    JWT_SECRET=
```

5. Start the server:
```npm start```
